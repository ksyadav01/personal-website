KARAN.GG


KARAN.GG is a stat puller and analyzer powered by Python and Riot Games' API. Using the Riot API wrapper "Riot Watcher" , KARAN.GG pulls relevant match history, user champ statistics, and much more for each command, and presents a concise table of information to the user.
